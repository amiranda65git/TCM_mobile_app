Comment pour installer les librairies et les dépendances ?
npm install

Commande pour lancer le test Android sur l'émulateur ou sur le téléphone. Nécessite Android Studio d'installer et un câble pour brancher le téléphone.
npx expo run:android

Commande pour lancer le build. APK.
npx eas build -p android --profile development
eas build -p android
